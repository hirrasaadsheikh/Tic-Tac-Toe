# Tick Tac Toe

Built with React and Socket.io

################################
   Multiplayer Tic Tac Toe App 
################################


################################
    Install the dependencies
################################

In Visual Studio Code terminal, write;

cd react-tic-tac-toe
npm install

################################
    Run the server side
################################

npm run server

################################
    Run the client side
################################

1) cd client
2) npm install 
3) npm start
